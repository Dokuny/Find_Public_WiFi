# [Find_Public_Wifi](https://www.notion.so/dokuny/Find-Public-Wifi-a7e28573a69c45bfac39ba39f9e10be4)

> 서울시 기준, 내 주변 공공 와이파이 조회 서비스입니다.

## 📜주요 기능 소개

- 현재 위치를 기반으로 주변 20개의 공공 와이파이 정보를 조회합니다.

- 조회 이력을 관리할 수 있습니다.
